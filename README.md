# 미션 - 숫자 야구


## 구현할 기능 목록

- generateRandomNumbers() : 3자리 랜덤 수 생성
- gameContinue : true이면 게임 계속 진행 / false이면 게임 종료
- checkGuess : answer와 비교해서 스트라이크 / 볼 / 낫싱 출력
- askContinue : 정답을 맞췄을 때 게임을 계속 진행할 것인지 질문
